# Super TypeScript Mario

A small Mario-style platformer built with TypeScript + Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Controls

- Arrow keys or A/D: move
- Space/Up/W: jump
- R: restart
- F: fullscreen toggle
